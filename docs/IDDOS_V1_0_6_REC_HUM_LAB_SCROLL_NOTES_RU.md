# idDOS v1.0.6 — REC hum + LAB scrollbar fix

## Исправлено

### 1. Гул после записи и сохранения
После сохранения записи аудиотракт теперь переводится в тихий idle-состояние:

- `M5Cardputer.Mic.end()`
- `M5Cardputer.Speaker.stop()`
- сброс внутренних флагов REC/playback

Это должно убрать остаточный фон/гул после выхода из REC/SAVE.

Дополнительно короткие системные звуки (`soundSave`, `soundSuccess`, `soundError`, `soundRun`) теперь останавливают Speaker после тона, чтобы усилитель не оставался в активном состоянии без сигнала.

### 2. LAB scrollbar
В LAB меню стрелки справа заменены на вертикальный scrollbar, как в других меню.

Теперь видно положение в списке LAB и понятно, что ниже есть `TOOLS`.

## Не трогалось

- BALL
- I2C logic
- файловая система
- команды shell
- REC/WAV формат

## Тест

1. Зайти в LAB → REC.
2. Записать короткий звук.
3. STOP → SAVE → имя → сохранить.
4. Проверить, исчез ли гул после сохранения.
5. Зайти в LAB и проверить scrollbar справа.
