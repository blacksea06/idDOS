# idDOS v1.0.9 Aesthetic Polish

База: `iddos_v1_0_8_sorted_keyboard_rollback_fix.zip`.

Это не ветка с шестерёнками. Это чистый рабочий движок v1.0.8 плюс аккуратная эстетическая полировка.

## Что изменено

### REC / диктофон

- Убраны лишние направляющие линии в зоне диктофона.
- Волна теперь рисуется как живая ECG-линия, без сетки и без верхней/нижней рамки.
- Добавлены noise gate, быстрый attack и более медленный release.
- Слабый голос должен давать более заметный отклик, а не маленькую попрыгушку.

### WAV player

- Плеер получил более олдскульный вид: `WAV DECK`, `[ PLAY ] / [ PAUSE ]`, тонкая шкала времени, маркер позиции, линейка громкости.
- При play/pause и изменении громкости теперь обновляется только динамическая зона, а не весь экран.
- Это должно убрать рябь/мигание от полного redraw.

### Welcome / prompt

- Текст приветствия не изменён.
- Если welcome открыт и пользователь просто нажимает OK/ENTER на пустой строке, приветствие остаётся на месте.
- Пустые `id:\>` больше не должны плодиться в столбик.
- Welcome исчезает только после ввода настоящей команды.

### Живая командная строка

- Нижний курсор `_` теперь мигает.
- Это показывает, что нижняя командная зона активна и система живая.
