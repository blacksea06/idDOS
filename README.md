# idDOS v1.0

idDOS — минималистичная операционная среда для M5Stack Cardputer-Adv, ориентированная на прямое взаимодействие с устройством, shell-интерфейс и встроенные аппаратные инструменты.

## Возможности

- Командная оболочка (Shell)
- Файловый менеджер
- Текстовый редактор
- Поддержка SD-карт
- Запись аудио
- Воспроизведение аудио
- WLAN Tools
- Bluetooth Lab
- Hardware Tools
- Breakout с управлением через IMU
- Загрузочный экран с диагностикой системы

## Аппаратная платформа

- M5Stack Cardputer-Adv
- ESP32-S3
- PlatformIO
